About the Project

The Waste Segregation Model is aimed at enhancing waste separation at the source for better recycling outcomes and a cleaner environment. The project focuses on using machine learning to classify waste into two categories: 'organic' and 'recyclable'. This classification helps in predicting whether an image contains organic or recyclable waste.

For more details, you can refer to the project proposal .

Key Features

  • Accurate waste classification using machine learning
  • Easy-to-use interface for uploading and predicting waste type
  • Detailed evaluation metrics for model performance
  • Option to retrain the model with new data
